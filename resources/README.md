# Free Cloud Security Resources

Everything here is free. No paywalls.

## Core reading

**AWS Security Best Practices**
https://aws.amazon.com/security/security-resources/

**CIS AWS Foundations Benchmark**
The standard for AWS security assessments. Free PDF download.
https://www.cisecurity.org/benchmark/amazon_web_services

**Cloud Security Alliance Security Guidance**
Comprehensive guidance on cloud security across providers.
https://cloudsecurityalliance.org/research/guidance

**NIST SP 800-210 — General Access Control Guidance for Cloud Systems**
https://csrc.nist.gov/publications/detail/sp/800-210/final

## Certifications worth knowing

**AWS Certified Security Specialty** is the gold standard for AWS security roles. It covers IAM, logging, infrastructure security and data protection.

**AWS Certified Cloud Practitioner** is the starting point. Good first step before the Security Specialty.

**CompTIA Cloud+** is vendor-neutral and covers cloud security concepts across providers.
